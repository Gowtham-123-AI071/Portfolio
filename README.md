# 3D Portfolio Website

A modern, glassmorphic, 3D-style portfolio built using React, Tailwind CSS, and Framer Motion.

## Tech Stack
- React + Vite
- Tailwind CSS
- Framer Motion
- Glassmorphism UI

## Features
- Responsive design
- 3D hover animations
- Glass cards
- Smooth transitions
